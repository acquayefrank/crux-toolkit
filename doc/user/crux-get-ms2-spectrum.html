crux-get-ms2-spectrum

Description:
Using a binary search, extract an MS-MS spectrum with a particular scan number from a .ms2 file, and output the spectrum. Optionally output some summary statistics for the MS-MS spectrum.
Usage: 
crux-get-ms2-spectrum [options] 
   <scan number> 
   <input filename> 
   <output filename>
Input:
  • <scan number> – The scan number for the MS-MS spectrum to extract from the ms2 file. This is an integer in the range [1, 100000], and uniquely identifies a particular MS-MS spectrum within a .ms2 file.
  • <input filename> – A file containing multiple MS-MS spectra in .ms2 format.
Output:
  • <output filename> – The name of the file in which to place the extracted MS-MS spectrum.
Options:
  • --stats – Output some summary statistics for the particular spectrum to STDOUT (not to the --ms2-out output file). Each stat is be placed on a separate line, preceded by it's name, followed by a colon, followed by the appropriate value (e.g. "TIC:1000.0\n"). The following stats are reported:
    • Precursor m/z
    • Total Ion Current
    • Base Peak Intensity
    • Number of peaks
    • Minimum m/z
    • Maximum m/z
    And for each possible spectrum charge state, output:
    • Charge state
    • Neutral mass
    • Charged mass
    • M+H+ mass
Warning messages:
  • Warning: The ms2 file "ms2 input filename" does not contain the spectrum with scan number "scan number".
