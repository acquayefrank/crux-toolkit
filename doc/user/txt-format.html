Txt file format

The txt file format records the matches between MS/MS spectra and a sequence database using lines of tab delimited fields.

A txt file consists of a single line containing the names of fields separated by tabs, followed by one or more lines giving the field values separated by tabs.

A sample txt file may be found here.

List of fields

  1. scan
  2. charge
  3. spectrum precursor m/z
  4. spectrum neutral mass
  5. peptide mass
  6. deta_cn
  7. sp score
  8. sp rank
  9. xcorr score
  10. xcorr rank
  11. -log(p-value)
  12. Weibull est. q-value
  13. percolator score
  14. percolator rank
  15. percolator q-value
  16. q-ranker score
  17. q-ranker q-value
  18. b/y ions matched
  19. b/y ions total
  20. matches/spectrum
  21. sequence
  22. cleavage type
  23. protein id
  24. flanking aa
  25. unshuffled sequence
