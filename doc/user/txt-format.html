Tab-delimited file format

Various programs in Crux report their output and read inputs in tab-delimited text format. Each such file consists of a single line containing the tab-separated names of fields, followed by one or more lines giving the corresponding field values.

Name Description search-for-matches barista q-ranker
PSM peptides proteins sub-proteins
1 scan The identifying number of each scan ✓ ✓ ✓     ✓
2 charge The charge state for this spectrum in this PSM. ✓ ✓ ✓     ✓
3 spectrum precursor m/z The observed m/z of the spectrum precursor ion. ✓ ✓ ✓     ✓
4 spectrum neutral mass The computed mass of the spectrum precursor at the given charge state. This is equal to the precursor m/z minus the mass of a proton (1.00727646677 Da), all multiplied by the charge. ✓ ✓ ✓     ✓
5 peptide mass The mass of the peptide sequence, computed as the sum of the the amino acid masses plus the mass of water (18.010564684 Da or 18.0153 Da, depending on whether we are using monoisotopic or average mass). ✓ ✓ ✓     ✓
6 delta_cn The normalized difference in xcorr for this PSM relative to other PSMs for this spectrum at this charge state. ✓ ✓ ✓     ✓
7 sp score The SEQUEST-type preliminary score. ✓ ✓ ✓     ✓
8 sp rank The rank of this PSM when sorted by sp score. ✓ ✓ ✓     ✓
9 xcorr score The SEQUEST-type cross correlation score. ✓ ✓ ✓     ✓
10 xcorr rank The rank of this PSM when sorted by xcorr. ✓ ✓ ✓     ✓
11 p-value The p-value computed from the estimated Weibull parameters of the score distribution for this spectrum. ✓         ✓
12 Weibull est. q-value The q-value computed from the p-value. ✓          
13 Weibull est. PEP The PEP computed from the p-value. ✓          
14 decoy q-value (xcorr) The empirical q-value computed by ranking target and decoy PSMs for this spectrum by xcorr. ✓          
15 decoy PEP (xcorr) The PEP computed by ranking target and decoy PSMs for this spectrum by xcorr. ✓          
16 percolator score he discriminant score assigned by percolator. ✓          
17 percolator rank The rank of this PSM when sorted by percolator score. ✓          
18 percolator q-value The q-value assigned by percolator. ✓          
19 percolator PEP The PEP assigned by percolator. ✓          
20 q-ranker score The discriminant score assigned by q-ranker.           ✓
21 q-ranker q-value The q-value assigned by q-ranker. The q-value is the minimal PSM-level false discovery rate at which this PSM is deemed significant. This q-value is computed based on the ranking of the PSMs induced by the Barista score.           ✓
22 q-ranker PEP The posterior error probability estimated from the q-ranker score           ✓
23 barista score The discriminant score assigned by barista.   ✓ ✓ ✓    
24 barista q-value The q-value assigned by barista.   ✓ ✓ ✓    
25 barista PEP The PEP assigned by barista   ✓ ✓      
26 b/y ions matched The number of b-ions and y-ions in the peptide that have a corresponding peak in the spectrum. ✓ ✓ ✓     ✓
27 b/y ions total The total number of b- and y-ions predicted for this peptide. ✓ ✓ ✓     ✓
28 matches/spectrum The number of candidate peptides in the database found for this spectrum. ✓ ✓ ✓     ✓
29 sequence The peptide sequence. ✓ ✓       ✓
30 cleavage type The cleavage rules for generating this peptide based on the user-specified enzyme specificity. ✓ ✓ ✓     ✓
31 protein id A comma-separated list of proteins in which this peptide appears. Following the protein name is a number in parentheses giving the start position of the peptide in the protein. ✓ ✓ ✓     ✓
32 flanking aa The amino acids that precede and follow this peptide in the parent protein ID. ✓ ✓ ✓     ✓
33 unshuffled sequence The target sequence from which a decoy was generated. ✓          
34 eta Estimated Weibull scale parameter. ✓          
35 beta Estimated Weibull shape parameter. ✓          
36 shift Estimated Weibull shift parameter. ✓          
37 corr The correlation of the scores with the estimated Weibull distribution. ✓          
38 RAW The raw (unnormalized) count of spectra per peptide computed by spectral-counts. ✓          
39 SIN A protein quantification score computed by spectral-counts. ✓          
40 NSAF The normalized spectral abundace factor computed by spectral-counts. ✓ ✓ ✓      
41 dNSAF The distributed normalized spectral abundace factor computed by spectral-counts. ✓          
42 EMPAI The exponentially modified protein abundance index computed by spectral-counts. ✓          
43 parsimony rank The protein rank based on its spectral-counts score. ✓          
44 decoy matches/spectrum The number of decoy peptides that were compared to this spectrum. May be different than the number of target peptides, depending on how the decoys were generated. ✓          
45 filename The name of the file in which the PSM appears.   ✓ ✓     ✓
46 psm_id PSM identifier.           ✓
47 protein group number A number that identifies a group.         ✓  
48 peptides-scan.charge Peptides belonging to each protein in the group, with scan and charge (for example, EAMPK-001285.2).       ✓ ✓  
49 proteins Proteins in the protein group. Each protein can be followed by peptides in parenthesis, if they belong only to this particular protein in the group. Since peptides are considered indistinguishable if they have identical amino acid sequences or they differ only by I/L or T/S in the same position in the peptide, the peptides shared by the group may not all be identical.       ✓ ✓  
50 parent group numbers The numbers of the protein groups in the barista.target.proteins.txt that are supersets of this group in terms of peptides.         ✓
