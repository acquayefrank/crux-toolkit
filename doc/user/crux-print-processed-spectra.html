crux print-processed-spectra

Usage:
crux print-processed-spectra <ms2 file> <output file>
Description:

For each spectrum in an ms2 file process the peaks as for computing xcorr and print out the transformed spectra in ms2 format.

Input:

  • <ms2 file> – The name of the ms2 file containing spectra to process
Output:
  • <output file> – The name of the ms2 file in which the processed spectra will be printed.
