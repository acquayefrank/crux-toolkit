crux compute-q-values

Usage:

crux compute-qvalues [options] <psm folder> <protein input>

Description:

Given a collection of scored PSMs, estimate q-values. The q-value is a statistical confidence measure that is analagous to a p-value but that incorporates multiple testing. The q-value associated with a score threshold T is defined as the minimal false discovery rate at which a score of T is deemed significant. In this setting, the q-value accounts for the fact that we are analyzing a large collection of spectra.

If the collection of PSMs contains both targets and decoys, then the false discovery rate associated with a given score is estimated as the number of decoy scores above the threshold divided by the number of target scores above the threshold.

Alternatively, if the collection of PSMs has p-values assigned by search-for-matches by using a --score-type of xcor_logp or sp_logp, then the false discovery rate associated with a given score is computed

Additional background information on q-values and statistical confidence measures can be found in this article:

Lukas Käll, John D. Storey, Michael J. MacCoss and William Stafford Noble. "Assigning significance to peptides identified by tandem mass spectrometry using decoy databases." Journal of Proteome Research. 7(1):29-34, 2008

Input:

  • <psm folder> – The name of the folder in which all the PSM result files are located. The program looks for files ending in .csm, which are produced by crux search-for-matches. All such files in the given directory are analyzed jointly.
  • <protein input > – The name of the file in fasta format or the directory containing the protein index from which to retrieve proteins and peptides.
Options:
  • --parameter-file <filename> – A file containing command-line or additional parameters. See the parameter documentation page for details.
  • --feature-file <string> – Optional file in which to write the features. Default = none.
  • --verbosity <int> – Specify the verbosity of the current processes from 0-100. Default = 40.
