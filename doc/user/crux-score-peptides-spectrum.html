crux-score-peptide-spectrum

Description:

Score a spectrum against a peptide list(peptides with charge), using the information in param and the specified scoring type. search-spectrum.

Usage:

crux-score-peptide-list-spectrum [options]  <peptide list file>  <scan number>  <ms2 file>

Input:

  • <list file> – The peptide list to score against the spectrum.

  • <scan number> – The scan number for the MS-MS spectrum to extract from the ms2 file. This is an integer in the range [1, 100000], and uniquely identifies a particular MS-MS spectrum within an .ms2 file.

  • <input ms2 filename> – A file containing multiple MS-MS spectra in .ms2 format.

Output:

    The program prints to standard output the score of each peptide vs. the spectrum, and optionally auxiliary information about how the score was generated.

Options:

  • [--score-type <sp>] – The type of scoring function to use. Default = 'sp'.

  • [--parameter-file <filename>] – The crux parameter file to parse parameter from. Default = Default to crux project defaults (to be determined).


Warning messages:

  • FATAL: The sequence "peptide" is not a valid peptide sequence.

  • FATAL: The value "charge" is not a valid charge state.

  • FATAL: The scan "scan-number" is not present in "ms2 input filename".

Bugs:

  • score-peptide-list-spectrum does not have bugs, it has features.
