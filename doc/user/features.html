Features used by percolator and q-ranker

Following is a list of the features used by the crux percolator and crux q-ranker commands to represent each peptide-spectrum match (PSM).

  1. XCorr: The SEQUEST cross-correlation score.
  2. DeltCN: The difference between this PSM's XCorr and the XCorr of the second-ranked PSM for this spectrum, divided by this PSM's XCorr. Note that this definition differs from that of the standard delta Cn reported by SEQUEST.
  3. Sp: The preliminary SEQUEST score.
  4. lnrSp: The natural logarithm of the rank of the match based on the Sp score
  5. Mass: The observed mass [M+H]+
  6. dM: The difference in calculated and observed mass
  7. absdM: The absolute value of the difference in calculated and observed mass.
  8. ionFrac: The fraction of matched b and y ions.
  9. lnSM: The natural logarithm of the number of database peptides within the specified m/z range
  10. enzN: Boolean: Is the peptide preceded by an enzymatic (tryptic) site?
  11. enzC: Boolean: Does the peptide have an enzymatic (tryptic) C-terminus?
  12. enzInt: Number of missed internal enzymatic (tryptic) sites.
  13. pepLen: The length of the matched peptide, in residues.
  14. charge1: Boolean: Is this a 1+ charged spectrum?
  15. charge2: Boolean: Is this a 2+ charged spectrum?
  16. charge3: Boolean: Is this a 3+ charged spectrum?
