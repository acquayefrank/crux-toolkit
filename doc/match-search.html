match-search

Description:

Given as input an ms2 file, a sequence database, and an optional parameter file, search each spectrum against the peptides in the sequence database, and return the resulting PSMs with some additional features in serialized binary output files or an SQT files.

Usage:

match_search  [options]  <ms2 input filename>  <protein input filename>

Input:

  • <ms2 > – The name of the file (in MS2 format) from which to parse the spectra.
  • <protein input filename> – The name of the file (in fasta format) from which to retrieve proteins and peptides.
Output:

    The program outputs to a user specified folder a binary file that contains the PSMs and additional features.
    Optionally, the result can be output in a human readable Sqt format file, or both binary and Sqt formats.
Options:
  • [--output-mode sqt|binary|all ] – The output format. Default = binary.
  • [--match-output-folder string] – The output folder to store the serialized binary output files. Default = current directory.
  • [--sqt-output-file] – Name of the sqt file to place matches. Default = Prefix of <ms2 input filename>.psm.
  • [--min-mass float] – The lowest spectrum m/z to search in the ms2 file. Default = 0.0
  • [--max-mass float] – The highest spectrum m/z to search in the ms2 file. Default = Positive Infinity.
  • [--prelim-score-type sp] – The type of scoring function to use. Default = sp.
  • [--score-type logp_exp_sp | logp_bonf_exp_sp | logp_evd_xcorr | logp_bonf_evd_xcorr | xcorrlogp_exp_sp | logp_bonf_exp_sp | xcorr] – The type of scoring function to use. Default = sp.
  • [--parameter-file filename] – The crux parameter file to parse parameter from. Default = Default to crux_parameter.
  • [--verbosity int] – Specify the verbosity of the current processes from 0-100. By default, verbosity is set to 100.
Warning messages:
  • Fatal: The file "spectra input filename" does not exist (or is not readable, or is empty).
  • Fatal: The file "protein input filename" does not exist (or is not readable, or is empty).
  • Fatal: The following options are incompatible:
    • --output-mode = sqt and --match-output-folder
    • --output-mode = binary and --sqt-output-file
Bugs:
  • match-search does not have bugs, it has features.
