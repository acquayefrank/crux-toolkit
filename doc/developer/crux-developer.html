Crux developers' documentation

The Crux developers' directory contains the following:

  • The Crux coding standards.
  • Information on how automake and autoconf are incorporated into crux. This includes information on how to build crux after checking out from the CVS repository.
  • Instructions on how to incorporate C++ into Crux.
  • The Crux API.
  • The Doxygen code documentation Run 'make doc' to produce it.
  • Notes on how new versions are released.
  • We keep a To Do list on Bugzilla.
