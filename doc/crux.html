Crux tandem mass spectrometry analysis software

In proteomics, the large-scale study of protein function and structure, one of the major challenges is to identify the proteins that are expressed in cells at different conditions. Mass spectrometry is a widely used method for protein identification; however, a detailed spectrum analysis is required to match the spectrum to the source protein. Crux contains software for tandem mass spectrometry analysis, with a focus on peptide identification.

The core programs in the Crux toolkit are

  • match-search: Search a collection of spectra against a sequence database, returning a collection of peptide-spectrum matches (PSMs).
  • match-analysis: Analyze a collection of PSMs using Percolator.

In addition, Crux contains the following utilities:

  • match-search Given as input an ms2 file, a spectrum scan number within that file, a sequence database, and an optional parameter file, search the spectrum against the peptides in the sequence database, and return high scoring peptides (replaces search-spectra).
  • predict-peptide-ions Given a peptide and a charge state, predict the ions according to the provided options.
  • get-ms2-spectrum Extract an MS-MS spectrum with a specified scan number from a .ms2 file.
  • score-peptide-spectrum Score a single spectrum against a single peptide.
  • score-peptide-list-spectrum Score a single spectrum against a given list of peptides.
  • search-spectrum Score a single spectrum against a given sequence database.
  • search-spectra Score a collection of spectra against a sequence database and return high scoring peptides.
  • spectacle Create an image file of an annotated spectrum.

The current version of Crux is 1.0

Crux is written in ANSI C for Unix/Linux platforms.

The Crux software was written by Chris Park and Aaron Klammer under the supervision of Prof. William Stafford Noble in the Department of Genome Sciences at the University of Washington, Seattle.
