generate-peptides

Description:

Given a protein fasta sequence database as input, generate a list of peptides in the database that meet certain criteria (e.g. mass, length, trypticity) as output.

Usage: 
generate-peptides [options] 
   <protein input filename>
Input:
  • <protein input filename> – The name of the file (in fasta format) from which to parse proteins.
Output:
  • STDOUT
    A series of header lines, describing how the peptides were generated, and according to what criteria. 
      # PROTEIN DATABASE: <protein input filename>
  # OPTIONS:  < options used to generate the file>
    Then a series of tab delimited lines with the following fields, where < > indicate an optional field. 
      mass \t protein-id \t peptide-start \t peptide-length <\t peptide-sequence> \n
Options:
  • [--output-sequence] – Output the peptide sequence as well as the protein id and start and stop.
  • [--min-mass float] – The minimum neutral mass of the peptides to output put. DEFAULT: 200
  • [--max-mass float] – The maximum neutral mass of the peptides to output. DEFAULT: 2400
  • [--min-length int] – The minimum length of the peptides to output. DEFAULT: 6
  • [--max-length int] – The maximum length of the peptides to output. DEFAULT: 50
  • [--tryptic | --non-tryptic ] – Mutually exclusive options describing the trypticity of the peptides to output. DEFAULT: tryptic peptides only.
    • --tryptic – indicates peptides that result from a complete in silico digestion of proteins after residues K or R except when followed by P.
    • --non-tryptic – indicates peptides that result from a complete in silico digestion of proteins after any residue.
  • [--sort-mass | --sort-length] – Mutually exclusive options governing how the output should be sorted. DEFAULT: sort by mass.
    • --sort-mass – sort the peptides by their mass.
    • --sort-length – sort the peptides by length.
Warning messages:
  • Fatal: The file "protein input filename" does not exist (or is not readable, or is empty).
  • Fatal: The file "peptide output filename" already exists (or cannot be created).
Bugs:
  • generate-peptides does not have bugs, it has features.
