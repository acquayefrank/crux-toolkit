msconvert

Usage with Tide:
msconvert --spectrumrecords <file [file...]>
Description:

This program is an extension of ProteoWizard's msconvert program, including support for writing Tide's spectrumrecords output format. In general, use msconvert as documented in the corresponding version at ProteoWizard, but use --spectrumrecords to convert files to the format used by Tide. The msconvert program distributed with Tide converts one or more files of tandem mass spectra from any ProteoWizard-supported format into a binary format that can be read by tide-search.

Input:
  • A file of tandem mass spectra in any of the formats supported by ProteoWizard. The input file names are given on the command line.
Output:
  • A binary file of spectra. The output file name is set automatically by msconvert by combining the input file name with the appropriate filename extension, which, for Tide-compatible files, is .spectrumrecords
Options:
Run ./msconvert for usage, options, and examples or see here for more.
Example for use with Tide:
./msconvert --spectrumrecords my-spectrum-file.mzML
